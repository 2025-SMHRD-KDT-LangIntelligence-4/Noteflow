Q1.
변수 선언 시 사용하는 'let'과 'const'의 가장 큰 차이점은?
정답: const는 재할당이 불가능하다. (O)
**[해설]** let은 재할당이 가능하지만, const는 선언 후 값의 재할당이 불가능하며, 주로 상수를 선언할 때 사용됩니다.
Q2.
자바스크립트에서 '===' 연산자는 값만 비교하고 자료형은 비교하지 않는다.
정답: 거짓 (X) / 내가 쓴 답: 참
**[해설]** '===' 연산자는 값과 **자료형**을 모두 엄격하게 비교합니다. 값만 비교하는 것은 '==' 연산자입니다.
Q3.
HTML에서 페이지 제목을 정의하는 태그는?
정답: <title> (O)
**[해설]** <title> 태그는 웹 페이지의 제목을 정의하며, 브라우저 탭에 표시됩니다.
Q4.
CSS에서 요소를 수평 중앙 정렬할 때 'margin: auto;'를 사용할 수 없는 경우는?
정답: display 속성이 flex일 때 (X) / 내가 쓴 답: display 속성이 block일 때
**[해설]** `margin: auto;`는 `width`가 지정된 **블록(block) 요소**에만 적용됩니다. `display: flex;` 상황에서는 `justify-content: center;`를 사용해야 합니다.
